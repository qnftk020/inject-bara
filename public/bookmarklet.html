3-Layer 다중 방어 3-Layer defense

정적 패턴 + 통계 시그니처 + LLM 판정. 어느 한 층이 뚫려도 다음 층이 잡습니다. Static patterns + statistical signatures + LLM judge. If one layer is bypassed, the next catches it.

L1

Static Scan

흰 글씨, display:none, zero-width 같은 시각적 은닉을 즉시 포착.Catches visual hiding like white text, display:none, and zero-width chars instantly.

→ Attack A: 시각적 은닉→ Attack A: Visual hiding
L2

PMI Signatures

"ignore previous..." 같은 인젝션 명령조 어휘를 코퍼스 학습으로 차단.Blocks injection command phrases like "ignore previous..." via corpus-trained signatures.

→ Attack B: 명령조 어휘→ Attack B: Command phrases
L3

LLM Judge

Gemini 기반 문맥 의미 분석 — 변칙 인젝션까지 최종 탐지.Gemini-powered semantic analysis — catches novel injections that bypass patterns.

→ Attack C: 신종 변칙→ Attack C: Novel variants
Defense in Depth. 한 레이어를 우회해도 다음 레이어가 방어합니다. Bypass one layer, the next one catches it.
