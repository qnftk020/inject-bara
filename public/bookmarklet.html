어떻게 작동하나

인젝바라, 세 가지 표정

북마크 클릭하면 우하단에 인젝바라가 등장. 클릭으로 깨우면 페이지를 스캔해서 인젝션을 찾습니다.

1

자고 있어요

북마크 클릭하면 우하단에 인젝바라가 자고 있는 상태로 등장합니다. 사이트는 평소대로 사용.

2

깨우면 스캔

인젝바라를 클릭하면 눈을 뜨고 페이지 전체를 스캔. 4가지 패턴(흰 글씨, 제로폭 문자, 의심 메타, 0px 글씨) 검사.

3

발견하면 경고

인젝션 발견 시 빨간 ! 표시 + 배지. 클릭하면 어떤 패턴이 어디 있는지 상세 패널 + "인젝션 숨기기" 버튼.

검출 엔진

3-Layer 다중 방어

정적 패턴 + 통계 시그니처 + LLM 판정. 어느 한 층이 뚫려도 다음 층이 잡습니다. 630개 공격 코퍼스와 5만 개 정상 문장으로 학습된 통계 모델이 단순 키워드 매칭을 넘어선 탐지를 제공합니다.

L1

Static Scan

정적 패턴 검사 — 흰 글씨, display:none, zero-width 같은 시각적 은닉을 즉시 포착합니다.

→ Attack A: 시각적 은닉
L2

PMI Signatures

통계 시그니처 — "ignore previous..." 같은 인젝션 특유의 명령조 어휘를 코퍼스 학습으로 차단합니다.

→ Attack B: 명령조 어휘
L3

LLM Judge

Gemini 기반 문맥 의미 분석 — 패턴/시그니처를 우회한 변칙 인젝션까지 최종 탐지합니다.

→ Attack C: 신종 변칙
Defense in Depth. 다층 구조로 단일 실패 지점(Single Point of Failure)을 제거합니다.
한 레이어를 우회해도 다음 레이어가 방어합니다.
